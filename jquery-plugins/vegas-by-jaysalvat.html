Vegas – Backgrounds and Slideshows


Vegas is a jQuery/Zepto plugin
to add beautiful backgrounds and Slideshows to DOM elements.

Important note: Vegas 2 is not a drop-in replacement for version 1.x. It's a brand new plugin.

Install

Download the last version.

Or use Bower:

bower install vegas

Or use NPM:

npm install vegas

Get started

First, include either jQuery or Zepto.

Then...

$(function() {
    $('body').vegas({
        slides: [
            { src: 'img1.jpg' },
            { src: 'img2.jpg' },
            { src: 'img3.jpg' }
        ]
    });
});

Contributing

Please don't edit files in the dist directory as they are generated via Gulp.
You'll find source code in the src directory!

Install dependencies.

npm install

Run watch task before editing code. 

gulp watch

Regarding code style like indentation and whitespace, follow the conventions you see used in the source already.
